# Mobile Shop

Mobile shop app created with React-Native,Typescript.

## Used Technologies:

- **React-Native**
- **Typescript**
- **expo** with **expo-notifications**
- **React Navigation**
- **React-Native-Reanimated for animations**
- **Formik** with **yup** for form validation

## Application Screens:

- Login
- Register
- Search
- Settings
- Cart
- Checkout
- Landing
- Reviews
- Purchase History
- Product Details
- Account
